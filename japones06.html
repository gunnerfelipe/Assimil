A Torre de Tóquio

1－Você conhece a Torre de Tóquio?
(Torre de Tóquio / [objeto] / saber / [pergunta])

2－Sim, eu conheço.
(sim / saber)

3－Como vou daqui?
(aqui / de / como / vai / [pergunta])

4－Primeiro, caminhamos até a estação de Meguro.
(primeiro / estação de Meguro / até / caminhar)

5－É perto.

6－De lá, vamos de trem até a estação de Shibuya.
(lá / de / estação de Shibuya / até / de trem / ir)

7－Então pegamos o ônibus na estação de Shibuya até a Torre.
(então / estação de Shibuya / de / Torre / até / ônibus / ir)

8－Na Torre há um aquário.
(Torre / [local] / aquário / [assunto] / existir)

9－Isso é interessante.

10－Existem também muitas lojas de souvenir.
(presente / [relação] / loja / também / muitos / existem)

Respostas

1－Estou indo para a Torre.

2－Eu cheguei.

3－Eu entrei.

4－Eu andei até a Torre.

5－Existem muitas lojas na Torre.

Notas
1－A Torre de Tóquio é muito parecida com a Torre Eiffel, embora seja mais alta por alguns metros, também é mais nova (construída em 1958). Turistas japoneses e estrangeiros se espremem para admirar a vista panorâmica do topo da Torre.

2－Meguro e Shibuya são as duas estações de trem de Tóquio, localizadas a poucos minutos uma da outra em uma das principais linhas, estão localizadas nos principais bairros de Tóquio.

3－Na Lição 5 (nas notas 2 e 3), vimos como escrever sho. sha é escrito da mesma forma: し shi com um pequeno: や ya = しゃ sha. Para as sílabas com sh, agora sabemos como escrever ações し shi, しょ sho e しゃ sha.
