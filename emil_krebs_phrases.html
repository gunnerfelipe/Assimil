Кто тебя учит русскому языку?

kto tʲebja uˈtɕit ˈruskəmʊ jɐˈzɨku

Quem está te ensinando russo?

  • Кто (Kto): Significa "quem" em russo. É um pronome interrogativo utilizado para fazer perguntas sobre pessoas.
  • тебя (tebya): É a forma do pronome "você" na forma acusativa. Indica que a pergunta está se referindo a "você".
  • учит (uchit'): É a forma do verbo "учить" (uchit) na terceira pessoa do singular, no presente. Significa "ensinar". Neste caso, está se referindo à ação de ensinar alguém.
  • русскому (russkomu): É o adjetivo "русский" (russkiy) no caso dativo. Significa "russo". Nesta frase, está no caso dativo para indicar a língua sendo ensinada.
  • языку (yazyku): É a forma do substantivo "язык" (yazyk) no caso dativo. Significa "língua". Nesta frase, também está no caso dativo para concordar com o adjetivo "русскому".

Где вы изучали русский язык?

[ɡdʲe vɨ ɪˈzut͡ɕɪlʲɪ ˈruskʲɪj jɐˈzɨk]

Onde você aprendeu russo?

  • Где (Gde): Significa "onde" em russo. É um advérbio de lugar utilizado para fazer perguntas sobre locais.
  • вы (vy): É o pronome "você" na forma de tratamento formal ou para se referir a mais de uma pessoa.
  • изучали (izuchali): É a forma passada do verbo "изучать" (izuchat'), que significa "estudar". Neste caso, está conjugado na terceira pessoa do plural para concordar com "вы" (você).
  • русский (russkiy): É o adjetivo "русский" (russkiy) que significa "russo". Nesta frase, está no caso acusativo para indicar o objeto direto do verbo "изучали".
  • язык (yazyk): É o substantivo "язык" (yazyk) que significa "língua". Neste caso, está no caso acusativo para concordar com o adjetivo "русский".
